# Spicy Snake
a snake game written in python


### I Installation

    pip install -r requirements.txt

On Windows, also do:

    pip install -r wondows_requirements.txt

### Usage

Run the game with 

    python spicy_snake/snake.py 

### Testing

Run the test with

    pytest

or 

    pytest -v

Calculate the coverage with

    pytest --cov
